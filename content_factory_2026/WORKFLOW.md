# WORKFLOW Quotidien + QA

## Routine quotidienne (45-90 min)
1. Choisir 1 niche + 1 angle
2. Générer 3 hooks + garder le meilleur
3. Produire script 30-45 sec
4. Générer voix + vidéo brute
5. Générer sous-titres + traduction FR/EN
6. QA finale + export
7. Publier + log performance

## Checklist QA obligatoire
- Hook clair dans les 2 premières secondes
- Une promesse + une preuve + un CTA
- Sous-titres lisibles (2 lignes max)
- Audio intelligible, pas saturé
- Format vertical 1080x1920
- Durée cible: 20-45 sec
- Description + hashtags alignés niche

## SLA production
- 1 vidéo/jour minimum
- 5 vidéos/semaines idéal
- Recyclage: 1 script = 3 variantes hooks

## Boucle d'amélioration hebdo
- Top 3 vidéos (watch time / saves)
- Identifier patterns gagnants
- Supprimer ce qui sous-performe 2 semaines de suite
